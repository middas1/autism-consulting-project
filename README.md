# Autism. One World for All — Consulting Project

**Strategic recommendations for an ASD support program, prepared for the Bulat Utemuratov Foundation**

## Overview

A structured consulting engagement supporting a foundation-run program for individuals with Autism Spectrum Disorder (ASD). The project involved gathering stakeholder input, analysing program performance, and delivering strategic recommendations to strengthen the program's long-term impact.

## Approach

- Gathered requirements and input directly from program stakeholders
- Analysed existing program performance data to identify strengths and gaps
- Delivered strategic recommendations covering three areas: program **sustainability**, **scaling** to reach more beneficiaries, and **post-18 support** — a stage where structured support for individuals with ASD typically drops off significantly

## Outcome

Produced a set of actionable, stakeholder-informed recommendations aimed at helping the foundation extend the reach and long-term viability of its ASD support program.

## Tools & Methods

Stakeholder interviews, program performance analysis, strategic consulting frameworks
